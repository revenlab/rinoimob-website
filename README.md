# Rinoimob Website

Nuxt 3 SSR - Public-facing real estate listing website.

## Tech Stack
- Nuxt 3
- Vue 3
- TypeScript
- TailwindCSS
- Server-Side Rendering (SSR)

## Getting Started

```bash
npm install
npm run dev
```

## Project Structure
- `pages/` - Nuxt pages
- `components/` - Vue components
- `layouts/` - Page layouts
- `server/` - Server API routes
- `content/` - Markdown content (blog, etc.)

## Features
- Property Listings
- SEO Optimization
- Blog/News
- Lead Capture
- Mobile Responsive
